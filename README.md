# Task Management System - Authentication Module

This project implements a secure authentication system using Spring Boot and JWT (JSON Web Token).

It provides user registration, secure login, password encryption, token-based authentication, and logout handling with token invalidation.


## Technologies Used

- Java 17+
- Spring Boot
- Spring Security
- JWT (io.jsonwebtoken)
- Maven
- MySQL
- Lombok

## Project Objective

This module was developed as part of an internship project to demonstrate secure backend authentication using industry-standard Spring Security and JWT practices.
